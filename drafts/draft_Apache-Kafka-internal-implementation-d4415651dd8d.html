Apache Kafka internal implementation

Apache Kafka is a messaging queue to achieve asynchronous communication between the applications.

Compared to other MQ technologies Kafka provides the following advantages:

  • It is a distributed platform
  • High performance
  • The retention period for a message is policy-based
  • works on the publisher-subscriber model
