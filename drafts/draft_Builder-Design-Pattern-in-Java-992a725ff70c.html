Builder Design Pattern in Java

Consider the House class, we need to create a builder for that class. It has 2 fields roof, floor, and door. Let us say, we need an object of House class with any of these fields initialized. Using builder class, we call corresponding methods to set values followed by build() method that returns object of House class setting all required values.


House.java

package com.example.basiccrudAPIs.builder;

public class House {
    private String roof;
    private String floor;
    private String door;

    public String getRoof() {
        return roof;
    }

    public void setRoof(String roof) {
        this.roof = roof;
    }

    public String getFloor() {
        return floor;
    }

    public void setFloor(String floor) {
        this.floor = floor;
    }

    public String getDoor() {
        return door;
    }

    public void setDoor(String door) {
        this.door = door;
    }

    @Override
    public String toString() {
        return "House{" +
                "roof='" + roof + '\'' +
                ", floor='" + floor + '\'' +
                ", door='" + door + '\'' +
                '}';
    }
}


HouseBuilder.java

package com.example.basiccrudAPIs.builder;

public class HouseBuilder {
    House house;

    public HouseBuilder() {
        house = new House();
    }

    public HouseBuilder setRoof(String roof) {
        house.setRoof(roof);
        return this;
    }

    public HouseBuilder setFloor(String floor) {
        house.setFloor(floor);
        return this;
    }

    public HouseBuilder setDoor(String door) {
        house.setDoor(door);
        return this;
    }

    public House build() {
        return house;
    }

}


Main.java

package com.example.basiccrudAPIs.builder;

import java.net.URISyntaxException;

public class Main {

    public static void main(String[] args) {
        House house = new HouseBuilder()
                .setRoof("Concrete")
                .setFloor("Tiles")
                .setDoor("Wooden")
                .build();
        System.out.println(house);
    }
}
